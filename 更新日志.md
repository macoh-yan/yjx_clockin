
# 更新日志

## [v1.0.0] - 2026-06-22

### 新增功能

#### 设备绑定机制
- 实现设备绑定功能，增强账号安全性
- 登录时自动检查设备绑定状态
- 支持自动绑定新设备
- 账号绑定其他设备时拒绝登录，防止账号被盗用

#### 时间规则与打卡优化
- 新增时间规则数据类 `PunchTimeRule`，管理打卡截止时间、早退阈值等规则
- 新增早退检测与确认功能，下班打卡时若未到下班时间需确认
- 新增防重复打卡机制，避免重复提交

#### 工具类
- **Constants** - 统一管理应用级常量（API配置、SharedPreferences key、权限请求码等）
- **DeviceUtils** - 设备相关工具类，提供设备ID获取和验证功能
- **ImageUtils** - 图片处理工具类，提供Base64解码、Bitmap转换、图片缩放等功能

### 安全性改进

1. **设备绑定**：强制账号与设备绑定，防止账号在非授权设备上使用
2. **虚拟定位检测**：打卡时检测虚拟定位，防止作弊打卡
3. **权限管理**：优化定位权限和相机权限的请求流程

### 用户体验优化

1. 对话框样式统一美化
2. 头像加载使用统一的图片处理工具
3. 个人中心显示设备绑定状态
4. 早退打卡时提供明确的确认提示
